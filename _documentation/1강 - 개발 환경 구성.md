# 개발 환경 구성 (윈도우)

## Chocolatey 설치

https://chocolatey.org

## Node.js 설치

choco를 이용해 설치합니다.

```
choco install nodejs.install
```

## Yarn 설치

choco를 이용해 설치합니다.

```
choco install yarn
```

# 개발 환경 구성 (맥)

## Xcode 설치

App Store 에서 검색 후 설치  
첫 실행 후 약관 동의 필수

## Homebrew 설치

https://brew.sh

```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## Node.js 설치

brew를 이용해 설치

```
brew install node
```

## Yarn 설치

```
brew install yarn
```
